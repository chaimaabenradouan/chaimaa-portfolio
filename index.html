Fraud Detection System

Developed a machine learning system for detecting fraudulent banking transactions using anonymized PCA-based features. Addressed severe class imbalance with SMOTE and conducted in-depth exploratory data analysis (EDA) to identify temporal fraud patterns.

Benchmarked multiple models including XGBoost, LightGBM, and Random Forest, and optimized performance using Recall and PR-AUC to improve fraud detection while minimizing false negatives. The final model was serialized using Joblib for deployment readiness.

Technologies: Python, Scikit-learn, XGBoost, LightGBM, Imbalanced-learn, Seaborn
